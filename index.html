Recommended Permanent Home

Keep snapshots separate from source code

This repository is the durable home for ZeroClaw metrics. The collector runs from scripts/build_dashboard.py, writes immutable snapshots under data/snapshots/, refreshes data/latest.json, derives data/daily.json and data/metrics.sqlite, and publishes this static index.html through GitHub Pages. The main application repository should link here rather than storing metrics snapshots in product source.

Collector: scripts/metrics Snapshots: separate metrics store SQLite: data/metrics.sqlite Dashboard: GitHub Pages /metrics Docs link: maintainer docs
